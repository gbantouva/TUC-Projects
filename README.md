# TUC Projects

A collection of projects implemented during my studies at **Technical University of Crete (TUC)**, School of Electrical and Computer Engineering.


## Overview

This repository contains various academic projects spanning multiple courses, programming languages, and technologies. The projects demonstrate skills in software development, systems programming, hardware design, web development, and cybersecurity.

## Projects by Course

### Autonomous Agents
**NAO Robot as an Italian Teaching Assistant**
- **Language**: Python
- **Technologies**: Webots simulation, Google Speech Recognition, Pyttsx3
- **Features**: Voice-controlled language learning, vocabulary and grammar lessons, interactive quizzes, cultural education

### Databases (PLH 303)
**University Department Database System**
- **Language**: SQL, PL/pgSQL
- **Database**: PostgreSQL
- **Features**: Student management, course registration, grading system, thesis assignment, triggers and stored procedures


### Data Structures and Files
**File Organization & Binary Search Trees**
- **Language**: Java
- **Topics**: Sequential files, sorted files, binary search trees, threaded binary search trees
- **Focus**: Search efficiency, file I/O, performance analysis

### Structured Programming (PLH 102)
**Electronic Prescription Management System**
- **Language**: Java
- **IDE**: Eclipse
- **Features**: Patient/doctor/medicine registration, prescription creation, search and reporting, cascading deletion

### Operating Systems
**TinyOS v.3**
- **Language**: C (C11 standard)
- **Platform**: Linux
- **Features**: Multicore preemptive scheduler, Unix-like process model, thread synchronization
- **Tools**: GCC, GDB, Valgrind, Doxygen

### Security Systems
**Cybersecurity Tools and Exploits**
- **Languages**: C, Python, Bash, Assembly
- **Libraries**: GMP, OpenSSL, libpcap
- **Projects**:
  - Cryptographic algorithms (Diffie-Hellman, RSA)
  - Access control logging system (LD_PRELOAD)
  - Web vulnerabilities (SQL injection, XSS)
  - Network traffic analyzer (packet sniffing)
  - Adblocker (iptables)
  - Intrusion detection (Snort IDS)
  - Buffer overflow exploitation

### System Design
**Team Building Event Planning System**
- **Language**: Java
- **Architecture**: MVC (Model-View-Controller)
- **Design Patterns**: Factory, Observer, Strategy
- **Features**: Event management, booking system, customer management

### Web Design
**Sexally - Sex Education Platform**
- **Technologies**: HTML5, CSS3, JavaScript
- **Features**: Educational articles, interactive quizzes, community chatrooms, responsive design
- **Theme**: Black and pink color scheme

### Computer Organization
**Processor Design and Implementation**
- **Language**: VHDL
- **Tools**: Xilinx ISE 14.7
- **Projects**:
  - Single-cycle processor
  - Multi-cycle processor with FSM
  - Pipelined processor with hazard detection
- **Components**: ALU, Register File, Control Unit, Forwarding/Stalling Units
